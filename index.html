Digital Admin

Developed a robust and scalable web application using .NET Core 8, deployed on Microsoft Azure, with SQL Server as the backend database. The application follows modern architecture practices and provides secure, high-performance data processing with cloud hosting and monitoring.

UCCPlus

Developed a full-stack web application using Angular for the frontend and ASP.NET Core for the backend, hosted on Microsoft Azure, with SQL Server as the primary database. Plus Digital

Developed a scalable and secure cloud-based application using ASP.NET Core as the backend, hosted on Microsoft Azure. The solution uses Azure Functions for serverless and background processing, Azure Key Vault for secure secrets management, Redis for high-performance caching, SQL Server for relational data storage, and Azure DevOps for CI/CD automation. The system is designed for high availability, performance, and enterprise-grade security.
